### Docker
Из hw02 запустить

`docker build -t hw02 .`

`docker run -ti -v {absolute path to artifacts}:/artifacts --name hw02 hw02`

Пример:

 `docker run -ti -v /home/karina/Python_HW2022/hw02/artifacts:/artifacts --name hw02 hw02`
